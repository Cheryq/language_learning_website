Zero To Hero

这是一个帮助人们从零开始学习世界各种语言的学习性网站，想要进行多种语言学习的同学可以在这个网站上得到很多的帮助。

该网站截至目前涵盖包括英语、日语、韩语、德语、法语、俄语、意大利语、西班牙语、葡萄牙语、阿拉伯语等在内的多种语言。所有语言都建立了常用单词库，帮助用户进行简单的单词记忆；每一种语言都有对应的书籍库与电影库，帮助用户在娱乐中学习。

此外，网站还提供了翻译的接口，当不明白单词与句子的意思时，可以进行快速的翻译、查找；我们还为每一个用户建立了独立的私人库，可以将单词，电影收录其中；更多的功能还有待去发现……
